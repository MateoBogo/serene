À propos du projet

Serene est une application réalisée dans le cadre d'un cours sur Ionic. Le but n'est pas de remplacer une vraie app de méditation, mais de comprendre les pages, les routes et les composants Angular.

La version actuelle est volontairement simple : elle sert de base pour ajouter ensuite le minuteur, le stockage local et les statistiques.
